# HuewAutomation
## General info

Automation Testing assignment for [Huew](https://huew.co) using java selenium tool.

## Language

- Java

## Tools

- Selenium Web driver

## Prerequisites

- Java jdk v1.8
- Chrome driver (https://chromedriver.storage.googleapis.com/index.html?path=2.45/), make sure the chrome driver is in the executable path.

## Running the automation test

- Download and extract the zip file. The folder contains a jar file and a java class file and the photo.
- Open Terminal and locate the folder
- Run the jar file using `java -jar <filename>.jar`
- The jar file prompts for email and password for google login and location of the image file to be uploaded. Enter the details and hit ENTER
- The automation test will now launch using Chrome web driver.
